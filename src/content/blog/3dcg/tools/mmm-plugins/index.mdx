MikuMikuMoving 用のプラグインの詰め合わせです。
以下のプラグインが含まれています。

- AnimateCaptionPlugin / 字幕アニメーション (Version 0.3)
  字幕の座標や角度などをアニメーションできます。
- ApplyNoisePlugin / ノイズ付加 (Version 0.5)
  選択したキーフレームにランダムな動きを付加します。
- ApplyOffsetPlugin / オフセット付加 (Version 0.8)
  選択したキーフレームに任意の移動量や回転量など加算します。
- ApplyRotationPlugin / 任意中心回転付加 (Version 0.8)
  選択したキーフレームのモーションを任意の方向に変更します。
- BeatBasedCopyPlugin / ビートコピー (Version 0.4)
  選択したキーフレームを BPM を指定して繰り返します。
- DetailedStatusPlugin / 詳細情報表示 (Version 0.5)
  選択したボーンやカメラの移動量と角度を小数点以下 6 桁まで表示します。
- DetailedTimePlugin / 詳細時間表示 (Version 0.3)
  現在のフレームの時間のミリ秒、小節、拍、Tick を表示します。
- GetMmdTransformationPlugin / MMD ポーズ取得 (Version 0.7)
  MMD からモデルの変形状態を取り込みます。
- RevertFramesPlugin / フレーム時間反転 (Version 0.3)
  選択したキーフレームを逆再生するように逆順にします。
- SetMmdTransformationPlugin / MMD ポーズ設定 (Version 0.5)
  MMD に現在の変形状態やモーションを送ります。
